To run Playwright tests on TestMu AI, define the `browserName`, `browserVersion`, and `platform` capabilities in your automation scripts.

## Browser and OS Capabilities
---

| Key | Expected Values | Description |
|-----|----------------|-------------|
| `browserName` | Chrome, Microsoft Edge, pw-chromium, pw-webkit, pw-firefox | Browser to test on |
| `browserVersion` | Chrome 83+, Edge 83+ | Browser version to test on |
| `platform` | **Windows**: 11, 10, 8, 8.1, 7. **macOS**: Monterey, Big Sur, Catalina, Mojave | Platform name |

```js
const capabilities = {
  'browserName': 'Chrome',
  'browserVersion': 'latest',
  'LT:Options': {
    'platform': 'Windows 10',
    'user': process.env.LT_USERNAME,
    'accessKey': process.env.LT_ACCESS_KEY,
  }
}
```
