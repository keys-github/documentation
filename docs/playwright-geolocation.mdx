Run your Playwright tests across 53+ geographies using the `geoLocation` capability. This is useful for testing location-based content such as pricing, language, or region-specific features.

## Configuration
---

```js
const capabilities = {
  'LT:Options': {
    'geoLocation': 'US',
  }
}
```

| Key | Values | Description |
|-----|--------|-------------|
| `geoLocation` | Country code (e.g., `US`, `GB`, `DE`, `IN`) | Simulate traffic from the specified country |
