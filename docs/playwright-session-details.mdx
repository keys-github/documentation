When you run a Playwright test on TestMu AI, the platform generates logs for each test session. Use the `getTestDetails` action to retrieve session information from within your test script.

## Retrieve Session Details
---

```js
let response = JSON.parse(await page.evaluate(_ => {},
  `lambdatest_action: ${JSON.stringify({ action: 'getTestDetails' })}`
))
console.log(response);
```
