After running your Playwright tests, view the results on the [TestMu AI Automation Dashboard](https://automation.lambdatest.com/).

## Available Information
---

The Automation Dashboard provides the following for each test session:

- **Video recording** of test execution
- **Console logs** from the browser
- **Network logs** for HTTP requests and responses
- **Screenshots** for each test step (when enabled)

To enable these logs, learn how to [configure debugging capabilities for Playwright tests](/support/docs/playwright-debug-tests/).

## Test Statuses
---

| Status | Description |
|--------|-------------|
| Completed | Test finished successfully |
| Failed | One or more assertions or steps failed |
| Stopped | Test was manually stopped |

To explicitly set test outcomes, learn how to [mark Playwright tests as passed or failed](/support/docs/playwright-mark-test-status/).
